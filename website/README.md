# RoboinvestingWebsite

## Frontend

### Installation
1. cd frontend
2. npm install

3. start server

```bash
npm run dev
```

## Backend

### Installation

1. Install node (at least v18.18.2) and npm
2. Install dependencies

```bash
npm install express mongoose dotenv cors body-parser
npm install nodemon --save-dev
npm install
```
3. Start server

```bash
npm run dev
```
